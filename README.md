# calcula-financiamento-veiculo

### Simulador de financiamento veicular
* Realiza o cálculo com 12x, 18x, 24x, 36x, 48x e 60x.
* Para realizar o cálculo, informa o valor do veículo, o valor da entrada e a taxa de juros.
